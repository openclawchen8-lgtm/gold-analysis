# 報告模板

根據報告類型選擇對應模板：

---

## 1. 驗證報告模板 (validation/)

```markdown
# TXXX - 驗證報告

**驗證者**: 樂樂  
**日期**: YYYY-MM-DD  
**任務**: TXXX - 任務名稱  
**狀態**: ✅ 通過 / ❌ 未通過

## 驗證項目

| 項目 | 預期 | 實際 | 結果 |
|------|------|------|------|
| 項目1 | | | ✅ |
| 項目2 | | | ❌ |

## 測試結果

```
測試輸出...
```

## 發現問題

1. 問題描述...
   - 建議: ...

## 結論

[通過/需修正]
```

- 建議2
